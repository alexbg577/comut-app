# Comut App

Application mobile de partage privé pour groupes (iOS/Android)

## Fonctionnalités

- 4 onglets: Publications, Shorts, Favoris, Compte
- Système de groupes privés
- Upload de photos, vidéos, musiques (max 3GB)
- Mode hors-ligne avec synchronisation
- Shorts style TikTok
- Commentaires, likes, favoris
- Gestion des membres (owner/admin/member)

## Build l'APK

```bash
cd comut-app
npx eas build --platform android --profile preview
```

Le build se fait dans le cloud, vous recevrez un lien de téléchargement.

## Variables d'environnement

L'app utilise le backend sur: https://comut-backend.onrender.com
